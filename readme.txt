== Description ==

Traduction française pour le plugin WP e-Commerce.
Ajoute certaines traductions manquante pour votre e-boutique


== Installation ==

1. Envoyer `wp-e-commerce-traduction-francaise` vers le dossier `/wp-content/plugins/`
2. Activer le plugin sur le menu 'Extensions' de WordPress
